### 项目介绍

本项目是在线学习的平台的后端服务。


### 运行环境

- 开发环境：IntelliJ IDEA
- 项目构建工具：Maven
- 数据库：MySQL
- JDK版本：jdk1.8


### 技术架构
- SpringBoot
- MyBatis
- Shiro
- JWT
- MySQL
- Redis


### 功能模块

CMS管理：用户登陆注册，信息增删改查

课程管理：课程增删改查，课程信息，课程上传删除，课程预览，课程搜索

学习中心：视频播放，课程收藏，用户笔记，数据分析
